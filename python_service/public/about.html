About the Project

Project Description

BhaavChitra is a comprehensive tool designed to analyze emotions and sentiments expressed in various forms of text, including social media posts, reviews, and general text entries. By leveraging advanced algorithms, this system provides users with insights into the overall sentiment of the input text.

Key Points

  • Real-time sentiment analysis for text and social media.
  • User-friendly interface for easy interaction.
  • Supports hashtags and URLs for social media analysis.

What It Uses

The system utilizes the following technologies:

  • VADER and BERT for sentiment analysis of social media and text.
  • JavaScript for frontend interactivity.
  • Flask and Python for backend processing.

What It Can Do

This system can perform various types of analyses, including:

  • Text Analysis: Analyzes any text input for sentiment classification.
  • Twitter Post Analysis: Fetches and analyzes recent tweets based on user-defined hashtags or links.

How It Is Better

The Text and Social Media Sentiment Analysis System offers several advantages over existing solutions:

  • Integrated analysis of both direct text and social media content.
  • User-driven insights with real-time feedback.
  • Cost-effective and efficient for personal and commercial use.
